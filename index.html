Introduction to Flexbox

CSS Flexbox is a powerful layout module that enables you to create flexible and responsive layouts with ease. It provides a more efficient way to design and biuld user interfaces, making it a fundamental skill for the web developers and designers.

What is Flexbox?

Flexbox, short for Flexible Box, is a CSS layout model that allows elements to adapt to different screen sizes, orientations and devices. It hepls you manage the layout, alignment, and distribution of elements within a container.

Why Use Flexbox

  • Responsive design: Flexbox makes it easy to create responsive layouts that adapt to multiple screen sizes.
  • Simplified layout management: Flexbox eliminates the need for complex float and positioning techniques.
  • Improved alignment and spacing: Flexbox provides precise control over element alignment and spacing.
  • Enhanced accessibility: Flexbox supports accessible layout practices.

Key Flexbox Concepts

Before diving deeper, let's get familiarized with these essential Flexbox terms:

  • Flex Container: The parent element that contains the flexible elements.
  • Flex Items: The child elements within the flex container.
  • Main Axis: The primary axis of the flex container (horizontal or vertical).
  • Cross Axis: The secondary axis perpendicular to the main axis.

Browser Support

Flexbox is supported by all major browsers, including Chrome, Firefox, Safari, Edge, and Internet Explorer (version 11 and above).

Basic Concepts
  1. Flex Container

    2. The flex container is a parent element that contains the flexible elements (flex items).

    To create a flex container, set disply:flex or display:inline-flex on the parent element.

    .container{ display:flex; }
  2. Flex Items

    3. Flex items are the child elements within the flex container.

    By default, flex items are laid out horizontally (row) or vertically (column)

  3. Axes

    4. Main Axis (horizontal or vertical): The primary axis of the flex container.

    Cross Axis (perpendicular to main axis): The secondary axis.

  4. Directions

    5. Flexbox supports two directions: Row (default):Items are laid out horizontally.Column: Items are laid out vertically.

    Example:

    .container { display:flex; flex-direction:row; or column; }
  5. Wrap

    6. Flex items can wrap to multiple lines: No Wrap (default): Items do not wrap i.e they do not break into the next line even when there's no more space within the container. Wrap: Items wrap into multiple lines when there is no more horizontal space within the container.

    Example:

    .container { display:flex; flex-wrap: wrap; }
Flexbox Properties
Justify content

Justify-content defines the distribution of flex items along the main axis.

Values:

  • flex-start (default)
  • center
  • flex-end
  • space-between
  • space-around
  • space-evenly

Example:

.container { display:flex; justify-content:center; }
Align items

The align-items property defines the alignment of flex items along the cross axis.

Values:

  • flex-start
    • center (default)
  • flex-end
  • baseline
  • stretch

Example:

.container{ display: flex; align-items: center; }
Flex direction

Flex-direction defines the direction of the flex items.

Values:

  • row (ddefault)
  • row-reverse
  • column
  • column-reverse

.container{ display: flex; flex-direction:column; }
Flex wrap

Flex-wrap defines whether or not, flex items wrap to multiple lines.

Values:

  • nowrap (default)
  • wrap
  • wrap-reverse

Example:

.container { display: flex; flex-wrap: wrap; }
Flex grow

Flex-grow is a property that defines the ability of a flex item to grow and occupy available space.

Values:

  • 0 (default)
  • Positive number (e.g., 1, 2, 3, 4, ... )

Example:

.item { flex-grow: 1; }
Flex shrink

Flex-shrink defines the ability of a flex item to shrink when space is limited.

Values:

  • 1 (default)
  • Positive numbers (e.g., 0, 1, 3,...)

Example:

.item { flex-shrink:0; }
Flex basis

This property defines the initial width or height of a flex item.

Values:

  • Auto (default)
  • Length units (e.g., px, %, em)

Example:

.item{ flex basis: 200px; }
Gap

Gap defines the spacing between flex items.

Values:

  • 0 (default)
  • Length units (e.g., px, %, em)

Example:

.container{ display: flex; gap: 20px; }
Align content

Align-content property defines the alignment of multiple lines of flex items.

Value:

  • flex-start
  • center
  • flex-end
  • space-beween
  • space-around
  • space-evenly

Example:

.container{ display: flex; flex-wrap:wrap; align-content:center; }
